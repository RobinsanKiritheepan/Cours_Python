# PowerPoints — Machine Learning

**Un deck par séance** — idéal pour réviser avant un TP ou un examen.

| Deck | Séance | Contenu |
|---|---|---|
| `02_S1_Regression_Lineaire.pptx` | S1 | EDA, moindres carrés, train/test, MAE & RMSE |
| `03_S2_Regression_Multiple_et_Classification.pptx` | S2 | coefficients, sigmoïde, matrice de confusion |
| `04_S3_KNN_NaiveBayes_Desequilibre.pptx` | S3 | KNN, Naive Bayes, classes déséquilibrées |
| `05_S4_Arbres_de_Decision.pptx` | S4 | Gini, profondeur, importances, frontière de décision |
| `06_S5_Multiclasse.pptx` | S5 | matrice 4×4, sélection de features |
| `07_S6_Forets_GridSearch_SHAP.pptx` | S6 | Random Forest, GridSearch, SHAP |
| `08_S7_Boosting_ROC_AUC.pptx` | S7 | Gradient Boosting, courbe ROC & AUC |
| `09_S8_Clustering.pptx` | S8 | KMeans vs DBSCAN, coude & silhouette |
| `10_S9_Non_supervise_reel.pptx` | S9 | clustering sur données réelles |

## Structure de chaque deck

1. Une **page de titre** rappelant les notebooks concernés
2. Les **diapos de cours** (concept → pourquoi → lecture des résultats)
3. Une diapo **« À retenir »** : les 4 idées clés + 🔴 **les pièges de la séance**
