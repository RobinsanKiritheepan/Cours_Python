# Cours Python & Intelligence Artificielle

Notebooks, exercices et supports de cours — des bases de Python jusqu'au Machine Learning.

## 📂 Contenu

### `jupyter_notebook/` — Les bases de Python
`day1` → `day5` : types, opérateurs, structures de données, fonctions, POO, décorateurs, matplotlib et pandas. Le dossier `Complet/` contient les versions détaillées en français.

### `Cours_IA/` — Machine Learning
Progression `S1` → `S8` :

| Séance | Thème |
|--------|-------|
| S1 | Régression linéaire simple, EDA, MAE/RMSE |
| S2 | Régression multiple, première classification (régression logistique) |
| S3 | KNN, Naive Bayes, classes déséquilibrées |
| S4 | Arbres de décision, importance des variables |
| S5 | Classification multiclasse, sélection de features |
| S6 | Forêts aléatoires, GridSearch, SHAP |
| S7 | Risque de crédit : Random Forest / Gradient Boosting, courbe ROC & AUC, SHAP |
| S8 | Clustering non supervisé : KMeans, DBSCAN, PCA, silhouette |

Le dossier `Cours_IA/Complet/` contient les notebooks annotés (cours détaillé en français).

### `PowerPoints/`
Supports de présentation : un cours complet + un deck dédié par séance.

## 🛠️ Technologies
Python · pandas · NumPy · scikit-learn · matplotlib · seaborn · plotly · SHAP
